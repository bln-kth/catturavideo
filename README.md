# catturavideo
This is api scripts for https://catturavideo.com/

## To Run
* Install/Download python:
https://www.python.org/downloads/
* Install/Download pip
https://pip.pypa.io/en/stable/installing/
* Install deps:
`pip install jwt`
* Run the script:
`python test-jwt.py`
